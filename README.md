# PIXI.js 5 with Webpack 4

使用最新的webpack4搭建。

安装依赖

```
npm i
```

本地开发运行

```
npm run dev
```

打包编译

```
npm run build
```

编译产出的文件在dist目录下，你可以通过例如vscode的`live server`插件快速预览该项目。
